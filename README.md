# Chordal WIP

A work-in-progress Python application for generating chord progressions using music theory and Markov chains, with a Kivy-based GUI.

## Features

- Generate chord progressions in various modes (Ionian, Dorian, Phrygian, etc.)
- View progressions as triads or 7th chords
- Interactive GUI to select root notes, modes, and chord types
- Markov chain-based progression generation for musically interesting sequences
- Kivy GUI

## Run GUI (experimental)

Run `python -m chordal_wip.gui`
